จังหวัดตาก

จังหวัดตากตั้งอยู่ภาคเหนือตอนล่างของประเทศ มีพื้นที่ทั้งหมด 16,406.65 ตารางกิโลเมตร เป็นจังหวัดที่มีขนาดใหญ่เป็นอันดับ 4 ของประเทศและเป็นอันดับ 2 ของภาคเหนือ สูงกว่าระดับน้ำทะเล 116.2 เมตร มีระยะทางห่างจากกรุงเทพฯ เป็นระยะทาง 426 กิโลเมตร

สภาพภูมิประเทศของจังหวัดตากเป็นพื้นที่ตามแนวเทือกเขา โดยสภาพพื้นที่จะถูกแบ่งออกเป็น 2 ส่วนใหญ่ ๆ โดยมีเทือกเขาถนนธงชัยกั้นกลาง ทำให้ลักษณะภูมิอากาศแตกต่างกันไป เนื่องจากเทือกเขาถนนธงชัยเป็นตัวปะทะมรสุมตะวันตกเฉียงใต้ที่พัดมาจากมหาสมุทรอินเดียและทะเลอันดามัน ทำให้ซีกตะวันออกได้รับความชุ่มชื้นจากลมมรสุมไม่เต็มที่ สภาพแห้งแล้ง ส่วนฝั่งตะวันตกจะได้รับอิทธิพลจากลมมรสุมมากกว่าทำให้ปริมาณฝนตกมีความชุ่มชื้นโดยเฉพาะในที่ที่อยู่ในเขตภูเขา

ที่ตั้ง อาณาเขต

ส่วนที่ 1

ตั้งอยู่ด้านทิศตะวันออก ประกอบด้วย 4 อำเภอ ได้แก่ อำเภอเมืองตาก อำเภอบ้านตาก อำเภอสามเงา และอำเภอวังเจ้า สภาพพื้นที่ส่วนใหญ่เป็นภูเขาสูงและที่ราบสูง ลาดเอียงลงไปทางทิศตะวันออกสู่แม่น้ำปิงและแม่น้ำวัง ทำให้เกิดพื้นที่ราบแคบริมสองฝั่งของแม่น้ำ

ส่วนที่ 2

ตั้งอยู่ทางด้านทิศตะวันตก ประกอบด้วย 5 อำเภอ ได้แก่อำเภอแม่สอด อำเภอพบพระ อำเภอแม่ระมาด อำเภออุ้มผาง และอำเภอท่าสองยาง ภูมิอากาศได้รับอิทธิพลมาจากมหาสมุทรอินเดียและทะเลอันดามัน มีความชุ่มชื้น ฝนตกมากกว่าซีกด้านตะวันออก

จังหวัดตากมีอาณาเขตติดต่อกับจังหวัดแม่ฮ่องสอน เชียงใหม่ ลำพูน ลำปาง สุโขทัย กำแพงเพชร นครสวรรค์ อุทัยธานี กาญจนบุรี และสาธารณรัฐแห่งสหภาพเมียนมา ยาวประมาณ 540 กม. (แม่น้ำเมย 170 กม เทือกเขาตะนาวศรี 370 กม.)

อำเภอเมืองตาก

ประวัติศาสตร์ : เป็นเมืองเก่าแก่ริมแม่น้ำปิง เคยเป็นศูนย์กลางการค้าและการเดินทัพสำคัญในสมัยกรุงศรีอยุธยา

อาณาเขตติดต่อ : อำเภอบ้านตาก,วังเจ้า,พบพระ,แม่สอด,แม่ระมาด และจังหวัดสุโขทัย,กำแพงเพชร

จุดเด่นที่ห้ามพลาด : ในตัวเมืองตากมี ศาลสมเด็จพระเจ้าตากสินมหาราช ที่เป็นศูนย์รวมจิตใจของชาวเมือง และมี สะพานสมโภชกรุงรัตนโกสินทร์ 200 ปี (สะพานแขวน) ที่เป็นจุดชมวิวแม่น้ำปิงและอาทิตย์ตกดินที่สวยงาม

อำเภอบ้านตาก

ประวัติศาสตร์ : เป็นเมืองเก่าแก่ดั้งเดิมในสมัยสุโขทัย เชื่อว่าเป็นถิ่นกำเนิดของสมเด็จพระเจ้าตากสินมหาราช

อาณาเขตติดต่อ : อำเภอเมืองตาก,สามเงา,แม่ระมาด และจังหวัดสุโขทัย

จุดเด่นที่ห้ามพลาด : เจดีย์ยุทธหัตถี (เจดีย์เฉลิมพระเกียรติพระเจ้ารามคำแหงมหาราช) วัดพระบรมธาตุบ้านตาก: เป็นที่ประดิษฐานพระบรมธาตุที่จำลองแบบมาจากเจดีย์ชเวดากอง และเป็นที่ตั้งของ หลวงพ่อทันใจ ที่มีชื่อเสียงเรื่องความศักดิ์สิทธิ์ วนอุทยานไม้กลายเป็นหิน: สถานที่ที่พบไม้กลายเป็นหินยาวที่สุดในโลก (กว่า 72 เมตร)

อำเภอสามเงา

ประวัติศาสตร์ : สามเงาเป็นชุมชนเก่าแก่ริมแม่น้ำปิง มีบทบาทด้านการคมนาคมและการตั้งถิ่นฐาน

อาณาเขตติดต่อ : ติดต่ออำเภอบ้านตาก,แม่ระมาด และจังหวัดลำปาง,ลำพูน,เชียงใหม่

จุดเด่นที่ห้ามพลาด : เป็นที่ตั้งของเขื่อนภูมิพล ซึ่งเป็นเขื่อนคอนกรีตโค้งที่ใหญ่ที่สุดในประเทศไทย และมีเส้นทางล่องแพชมความงามเหนือเขื่อนไปจนถึงวัดพระธาตุแก่งสร้อย

อำเภอวังเจ้า

ประวัติศาสตร์ : ปรากฏชื่อในศิลาจารึกสุโขทัยว่าเป็นเมืองหน้าด่านสำคัญ และเคยเป็นที่ประทับพักแรมของพระมหากษัตริย์ในอดีตขณะเสด็จประพาส

อาณาเขตติดต่อ : ติดต่ออำเภอเมืองตาก,พบพระ,แม่ระมาด และจังหวัดกำแพงเพชร

จุดเด่นที่ห้ามพลาด : อำเภอวังเจ้าเป็นที่ตั้งของ อุทยานแห่งชาติคลองวังเจ้า ซึ่งมีแหล่งท่องเที่ยวทางธรรมชาติที่สมบูรณ์มาก โดยเฉพาะน้ำตกและจุดชมวิวเขาสันเย็น

อำเภอแม่สอด

ประวัติศาสตร์ : เป็นเมืองเก่าแก่ที่มีความสำคัญทางยุทธศาสตร์มาตั้งแต่สมัยอยุธยา และพัฒนาจากหมู่บ้านเล็กๆ สู่เขตพัฒนาเศรษฐกิจพิเศษและประตูการค้าสำคัญสู่เมียนมา

อาณาเขตติดต่อ : ติดต่ออำเภอเมืองตาก,พบพระ,แม่ระมาด,วังเจ้า และประเทศเมียนมา

จุดเด่นที่ห้ามพลาด : เป็นศูนย์กลางการค้าอัญมณีและสินค้าชายแดนที่ ตลาดริมเมย แล้ว แม่สอดยังมี สะพานมิตรภาพไทย-เมียนมา ที่เป็นจุดเชื่อมโยงเส้นทางระเบียงเศรษฐกิจตะวันออก-ตะวันตก (East-West Economic Corridor)

อำเภอพบพระ

ประวัติศาสตร์ : มีการตั้งถิ่นฐานของชนเผ่าหลากหลายเดิมชื่อ "บ้านเพอะพะ" เป็นภาษากะเหรี่ยงที่แปลว่า "มีน้ำซึมออกมาจากดิน" ก่อนจะเพี้ยนมาเป็นชื่อ "พบพระ" ที่มีความหมายเป็นมงคลในปัจจุบัน

อาณาเขตติดต่อ : ติดต่ออำเภอแม่สอด,อุ้มผาง,วังเจ้า จังหวัดกำแพงเพชร และประเทศเมียนมา

จุดเด่นที่ห้ามพลาด : เป็นแหล่งปลูกกุหลาบที่ใหญ่ที่สุดในประเทศไทย และมีน้ำตกพาเจริญ ที่มีชั้นหินลดหลั่นกันถึง 97 ชั้น

อำเภอแม่ระมาด

ประวัติศาสตร์ : เป็นชุมชนเก่าแก่ของชาวกะเหรี่ยงและละว้า เดิมเป็นส่วนหนึ่งของอำเภอแม่สอดก่อนจะแยกตัวออกมา

อาณาเขตติดต่อ : ติดต่ออำเภอท่าสองยาง,แม่สอด,บ้านตาก,สามเงา จังหวัดเชียงใหม่ และประเทศเมียนมา

จุดเด่นที่ห้ามพลาด : มีอุทยานแห่งชาติขุนพะวอ ที่มีอากาศเย็นสบายตลอดปี และมี วัดดอนแก้ว ซึ่งเป็นที่ประดิษฐานพระพุทธรูปหินอ่อนศิลปะแบบพม่าที่สวยงาม

อำเภออุ้มผาง

ประวัติศาสตร์ : เคยเป็นเมืองหน้าด่านและจุดตรวจหนังสือเดินทาง (อุ้มผะ) ของชาวกะเหรี่ยงที่เดินทางไปมาในอดีต ก่อนจะกลายเป็นอำเภอที่โดดเด่นด้านการอนุรักษ์ป่าไม้

อาณาเขตติดต่อ : ติดต่ออำเภอพบพระ จังหวัดกำแพงเพชร,กาญจนบุรี,อุทัยธานี และประเทศเมียนมา

จุดเด่นที่ห้ามพลาด : เป็นที่ตั้งของ น้ำตกทีลอซู ซึ่งเป็นน้ำตกที่ใหญ่ที่สุดและสวยที่สุดในประเทศไทย ตั้งอยู่ในเขตรักษาพันธุ์สัตว์ป่าอุ้มผาง ซึ่งได้รับการยกย่องให้เป็นมรดกโลกทางธรรมชาติร่วมกับเขตรักษาพันธุ์สัตว์ป่าทุ่งใหญ่นเรศวร-ห้วยขาแข้งด้วยครับ

อำเภอท่าสองยาง

ประวัติศาสตร์ : เดิมเป็นกิ่งอำเภอขึ้นกับจังหวัดแม่ฮ่องสอน ก่อนจะโอนมาขึ้นกับจังหวัดตากในปี พ.ศ. 2491 และได้รับชื่อตามหมู่บ้านที่มีต้นยางขนาดใหญ่สองต้นโดดเด่นอยู่ริมฝั่งน้ำเมย

อาณาเขตติดต่อ : ติดต่ออำเภอแม่ระมาด จังหวัดเชียใหม่,แม่ฮ่องสอน และประเทศเมียนมา

จุดเด่นที่ห้ามพลาด : ม่อนคลุยและม่อนทูเล : จุดชมทะเลหมอกและดาวยามค่ำคืนที่สวยงามและท้าทายสำหรับนักเดินป่า ถ้ำแม่อุสุ : ถ้ำหินงอกหินย้อยที่สวยงามและได้ชื่อว่าเป็น "โรงละครใต้ดิน" ขนาดใหญ่ วิถีชีวิตชาวปกาเกอะญอ: สัมผัสวัฒนธรรมท้องถิ่นและการแต่งกายที่เป็นเอกลักษณ์ของคนในพื้นที่
